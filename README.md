🌀 ContextNote.AI – Inteligencia Documental RAG

ContextNote.AI es una estación de trabajo avanzada diseñada para programadores y estudiantes que necesitan interactuar con grandes volúmenes de información. Utiliza una arquitectura RAG (Retrieval-Augmented Generation) para procesar múltiples archivos simultáneamente con precisión y privacidad.

✨ Características Principales:

🧠 Cerebro Antisaturación (Chunking): Fragmentación inteligente de documentos para procesar tesis, BOEs o códigos extensos sin errores de contexto.

📂 Soporte Multiformato: Lee y analiza archivos PDF, Word (.docx), Excel (.xlsx) y múltiples lenguajes de programación (.java, .py, .js, .ts).

🔗 Interacción Multi-archivo: Sube varios documentos a la vez; la IA cruzará datos entre ellos para darte respuestas integrales.

📍 Citaciones Exactas: Cada respuesta indica de qué archivo y sección proviene la información (Adiós a las alucinaciones).

🛡️ Privacidad Total (Botón de Pánico): Borrado físico de datos en Supabase con un clic al finalizar tu sesión.

💻 UI Matrix Edition: Interfaz optimizada con efecto de lluvia de código y modo oscuro profesional.
